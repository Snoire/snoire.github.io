关于公网 IP 和 私网 IP

即使不从事网络方面的工作，普通人也大概率会听说过这两个名词。但是，很少有人能够说清楚它们到底有什么区别。就比如我，之前一直稀里糊涂的，前些日子才搞明白。

互联网协议设计之初是这样考虑的，要为每一个联网的设备分配一个唯一的地址，根据这个地址就可以找到这台主机。至于怎么根据 IP 地址找到对应的主机，这一点暂时不需要考虑，我们只需要知道一点：只要一台主机有全球唯一的 IP 地址，那么这个 IP 地址就是和它绑定的，就像一个人的身份证号一样，知道你的身份证号，总可以找得到你。符合这样特点的 IP 地址，我们就叫它 公网 IP。

一开始是没有公网 IP 这个概念的，只是后来有了私网，普通的 IP 地址才被叫为公网。公网 IP 由于具有全球唯一性，所以不能想用就用，不然就乱套了。有专门负责分配 IP 地址的国际性组织，你想要 IP 地址，得去申请，通过了才能用。
