安装tar.gz(tgz)包

以tar.gz为扩展名的软件包，是用tar程序打包并用Gzip程序压缩的软件包。要安装这种软件包，需要先对软件包进行解压缩，使用 “tar －zxfv filename．tar．gz”可以对软件包进行解压缩，解压缩所得的文件在以Filename为名的目录中。进入该目录，可以看到解压缩出来的文件 了。各种软件都有不同的安装方法，但是一般每个软件包解压缩后都有Install和Readme文件，帮助文件中会有详细的安装指导。以 tar．gz（或tgz）包发行的软件有一个缺点，就是一般不带自动反安装程序，如果需要对已经安装的此类程序进行删除，就不得不仔细查看 Makefile 中的安装路径和文件名，这些对于初学者有一些难度。

  • tar -zxvf example.tar.gz
  • cd  example #检查readme文件，里面有介绍
    • 有些解压后直接可以使用
  • ./configure
  • make & make install

  1. 用管理员身份打开 cmd
  2. 输入 chcp 437
  3. 查看帮助 bcdedit /?

查看启动项

其中，Firmware Boot Manager/{fwbootmgr} 就是总的控制项

Windows Boot Manager/{bootmgr} 是 win10 的启动管理器

装的其他的启动管理器也会有自己的条目，比如 rEFInd Boot Manager 还有 Grub2 Boot Manager

所有的条目都用 identifier 来标识，就是用花括号括起来的那个，比如 {bootmgr} 或者 {619cffcb-5121-11e9-90da-806e6f6e6963} 这样子

修改启动项

主要是改 Firmware Boot Manager/{fwbootmgr} 里面的启动顺序，比如把 {bootmgr} 放在第一个，那么开机会直接启动 win10；把 rEFInd Boot Manager 对应的 identifier 放在第一，那么启动项将由 rEFInd 接管。

像这样，把 win10 放到最后一位，🤣

为目录建立软链接 win10创建软链接

为文件建立链接：

  • 建立软链接：类似于快捷方式

    mklink link_name target_name

  • 建立硬链接：和正常的文件几乎一模一样，也有文件大小，但是实际上不占用空间

    mklink /H link_name target_name

note: 如果文件名中有空格，就用英文双引号括起来
